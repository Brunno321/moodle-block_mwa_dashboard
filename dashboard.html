🚀 Início rápido
O MWA Dashboard transforma logs do Moodle em análises visuais de participação, engajamento e risco de evasão — sem instalar nada além do bloco.
Via bloco Moodle (automático)
Instale o plugin, adicione o bloco ao curso e clique em Acessar dashboard. Os dados são carregados automaticamente — sem upload manual.
Via upload manual (standalone)
1. Exporte os logs
Moodle → Administração → Relatórios → Logs → selecione o curso → Baixar em JSON
2. Exporte as notas
Moodle → Notas → Exportar → Planilha Excel (.xlsx) — opcional
3. Arraste os arquivos
Solte nas caixas de upload ou clique para selecionar
4. Clique Analisar
Botão ativa após carregar ao menos o JSON
💡O JSON sozinho habilita todas as análises de engajamento. O XLSX adiciona análise de notas, aprovados e a fórmula completa de participação.
✓Dados ficam só no seu navegador. Nenhum dado vai para servidores externos (exceto os recursos de IA).
📂 Carregar dados
Dois arquivos exportados diretamente do Moodle.
📋 JSON — logs de acesso
Contém todos os eventos: acessos, posts, envios de tarefa, visualizações de H5P. É o arquivo principal.
Como exportar
Administração → Relatórios → Logs → Filtrar por curso → Baixar → JSON
Campos usados
hora, nomecompleto, contextodoevento, componente, nomedoevento, email
📊 XLSX — livro de notas
Exportação padrão do Moodle. Habilita: nota média, aprovados/reprovados e fórmula completa de participação (nota 50% + cobertura 30% + entrega 20%).
⚠️XLSX e JSON precisam ser do mesmo curso para que os nomes dos alunos correspondam.
